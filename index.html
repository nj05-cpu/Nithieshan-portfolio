OCR Course Analyzer

An OCR processing pipeline built using Python, PyMuPDF, and Tesseract OCR to extract structured information from unstructured PDF course outline documents. The system converts PDFs into images, performs text extraction, and identifies key academic data such as course codes, evaluation breakdowns, and grading components. The pipeline is designed as a multi-stage workflow that handles diverse document formats, validates extracted text accuracy, and detects errors caused by irregular layouts or low-quality scans. Extracted results are returned in structured JSON format, enabling downstream analysis and automation. This project demonstrates applied computer vision concepts, data validation, and building robust pipelines for real-world document processing.
